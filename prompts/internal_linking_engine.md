# Internal Linking Engine Prompt

You are an SEO content strategist and expert in internal linking for websites.

Your task:
1. Take the following content structure for a genetic testing service in Hyderabad:

   - Pillar Page: "Genetic Testing in Hyderabad: Complete Guide (Cost, Types & Benefits)"
   - Cluster Pages:
     - Cluster 1: Cost of Genetic Testing in Hyderabad
     - Cluster 2: Types of Genetic Tests Explained
     - Cluster 3: How Accurate and Safe is Genetic Testing?
     - Cluster 4: Who Should Take Genetic Testing?
2. Suggest anchor text and placement for internal links:
   - Each cluster should link **back to the pillar page** where relevant  
   - The pillar page should link **to all cluster pages** at appropriate sections  
   - Suggest **cross-links between clusters** if it improves user experience
3. Ensure:
   - Links are natural and contextually relevant  
   - Avoid overstuffing links in any section  
   - Maintain readability  

---

## Output Format
- Provide a table or structured list showing:  
  - Source page  
  - Anchor text  
  - Target page  
  - Suggested placement (heading or paragraph)
